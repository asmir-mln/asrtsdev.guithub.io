## 1) Méthode de calcul
- Coût de revient HT (CRHT) = coûts directs + coûts indirects imputés
- Marge brute (%) = (PVHT - CRHT) / PVHT × 100
- Prix de vente HT (PVHT) = CRHT / (1 - marge cible)

## 2) Structure des coûts (modèle)
- Coûts directs :
  - Développement / production
  - QA / tests / validation
  - Infrastructure (serveurs, outils)
- Coûts indirects :
  - Gestion de projet
  - Conformité / administratif
  - Support / maintenance
