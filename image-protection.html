Protection des Droits d'Auteur

Mesures légales et techniques pour protéger vos créations

⚠️ Avertissement Légal

Toutes les images, illustrations et créations numériques présentes sur le site AsArt'sDev sont protégées par les droits d'auteur et droits connexes. Leur reproduction, distribution, vente ou utilisation sans autorisation écrite est strictement interdite et passible de poursuites judiciaires.

1. Protection Technique des Images

Le site utilise plusieurs mesures techniques pour protéger les images contre le vol :

  • Blocage du clic droit : Impossible de télécharger directement via le menu contextuel
  • Blocage du glisser-déposer : Pas d'export par drag and drop
  • Marquage filigrane : Watermark intégré invisible pour tracer les usages autorisés
  • Système d'authentification : Validation des sources et utilisateurs légitimes
  • Meta-données automatiques : Signature numérique intégrée aux images

2. Fondement Juridique

Code de la Propriété Intellectuelle (France) :

  • Article L. 111-1 : "L'auteur est le créateur de l'œuvre"
  • Article L. 122-4 : Reproduction et représentation interdite sans autorisation
  • Article L. 335-2 : Contrefaçon : amende jusqu'à 300 000€ et/ou emprisonnement

Directive Européenne 2006/115/CE : Harmonisation de la protection des droits connexes

3. Conséquences Légales de la Violation

L'utilisation, la reproduction ou la distribution non autorisée des images peut entraîner :

  • 🔴 Amende civile : 1000€ à 300 000€
  • 🔴 Dommages et intérêts : Jusqu'à 3 fois le prix de vente
  • 🔴 Pénalités pénales : Jusqu'à 2 ans d'emprisonnement
  • 🔴 Saisi du matériel : Stockage illégal confisqué
  • 🔴 Retrait du contenu : Ordre de suppression immédiate

4. Code de Protection Intégré

Voici le code JavaScript automatiquement appliqué à tous les contenus du site :

// Protection des images AsArt'sDev document.addEventListener('DOMContentLoaded', function() { // Blocage du clic droit sur les images document.querySelectorAll('img').forEach(img => { img.addEventListener('contextmenu', e => { e.preventDefault(); alert('Cette image est protégée par les droits d\'auteur ©AsArt\'sDev'); return false; }); // Blocage du drag and drop img.addEventListener('dragstart', e => { e.preventDefault(); return false; }); // Ajout de class pour CSS protection img.classList.add('protected-image'); // Ajout de meta-données img.setAttribute('data-protected', 'true'); img.setAttribute('data-owner', 'AsArt\'sDev'); }); // Enregistrement des tentatives de violation document.addEventListener('copy', e => { if (window.getSelection().toString().length > 0) { console.warn('Événement de copie détecté sur contenu protégé'); } }); });

CSS appliqué :

img.protected-image { -webkit-user-select: none; -moz-user-select: none; -ms-user-select: none; user-select: none; -webkit-user-drag: none; pointer-events: all; } img.protected-image::before { content: ''; background-image: repeating-linear-gradient( 45deg, transparent, transparent 35px, rgba(0,0,0,.05) 35px, rgba(0,0,0,.05) 70px ); }

À chaque visite, vous acceptez les présentes conditions via una pop-up de confirmation. Cette pop-up :

9. Durée de Protection

Durée légale : La protection s'étend pendant toute la vie de l'auteur + 70 ans après sa mort (droit d'auteur français)

Après la mort du créateur, ses ayants droit conservent la totalité des droits patrimoniaux et moraux.
