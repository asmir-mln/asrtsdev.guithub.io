## EN — Vision

## FR — Checklist de lancement pro
- Identité de marque cohérente sur toutes les pages.
- Pages clés prêtes: accueil, contact, commande, paiement, CGV.
- Encodage UTF-8 propre (accents et symboles corrects).
- Système de paiement prêt à configurer (Stripe + clés sécurisées).
- Protection des contenus visuels active.
- Documentation de support disponible.

## EN — Professional launch checklist
- Consistent branding across all pages.
- Core pages ready: home, contact, order, payment, terms.
- Clean UTF-8 text rendering (accents/symbols fixed).
- Payment system ready for secure Stripe setup.
- Visual content protection enabled.
- Support documentation available.

## FR — Actions immédiates
1. Créer `api/config.php` depuis `api/config.example.php`.
2. Ajouter les clés Stripe test/live.
3. Installer Composer puis `stripe/stripe-php`.
4. Tester `paiement.html` + `confirmation-paiement.html`.
5. Publier en HTTPS uniquement.

## EN — Immediate actions
1. Create `api/config.php` from `api/config.example.php`.
2. Add Stripe test/live keys.
3. Install Composer and `stripe/stripe-php`.
4. Test `paiement.html` + `confirmation-paiement.html`.
5. Deploy with HTTPS only.
